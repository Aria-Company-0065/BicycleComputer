Bicycle Computer

This application displays and records bicycle riding information.

[ Ride information ]
• Speed (Avg, Max)
• Ride Time
• Distance
• Odometer
• Pace（Avg、Max)
• Altitude（Max, Min)
• Elevagion gain
• Slope
• Calorie

[ Map function ]
• Route Search
• Map type (Normal, Satellite, Night)
• Map rotation (Course up, North up)
• Adding markers

[ History ]
• Statistics
• Riding route
• Speed and altitude charts
   


In addition to riding bike, this application is also useful for outdoor activities such as hiking, running and motorcycles.
